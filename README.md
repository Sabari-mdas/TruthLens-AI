Problem Statement

Digital misinformation is spreading rapidly through social media, news platforms, and AI-generated images. Users often struggle to distinguish between real and fake content, especially when manipulated images and misleading headlines are shared together.

TruthLens AI solves this problem by providing a single platform that can:

Detect fake or misleading text-based news
Analyze uploaded images for manipulation or deepfake-like characteristics
Generate a combined trust report from both text and image analysis
Store previous analyses in a searchable history dashboard


Project Description

TruthLens AI is a web-based misinformation detection platform.

The project contains four main modules:

Text Analysis
Users can paste any article, headline, or social media post.
The system analyzes the text and predicts whether it is likely real or fake.
It highlights suspicious phrases, provides a confidence score, reliability percentage, explanation, and trusted sources for verification.
Image Analysis
Users can upload an image or screenshot.
The system checks for signs of manipulation, AI-generated content, or simple deepfake characteristics.
It returns anomalies, a confidence score, likely generation method, and verification tips.
Combined Report
Combines the latest text and image analysis into one final trust score.
Generates a summarized verdict and reasoning timeline.
Allows users to copy, share, or download the final report.
History Dashboard
Stores previous analyses locally.
Lets users search, filter, and reopen old analyses.
What Makes It Useful
Combines both text and image verification in one place
Simple and beginner-friendly interface
Fast results with visual explanations
Useful for students, journalists, social media users, and fact-checkers

How Google AI Was Used

Google AI is integrated into TruthLens AI in multiple ways:

Gemini analyzes pasted news text and determines whether it contains misinformation or suspicious claims.
Gemini extracts suspicious phrases and generates a human-readable explanation.
Gemini helps estimate reliability and confidence scores.
For image analysis, Gemini is used to inspect uploaded images and identify possible manipulation, AI-generated artifacts, or deepfake indicators.
Gemini generates the combined summary report and final trust score by merging both text and image results.
Google AI Studio was used during development to test prompts, refine outputs, and improve the quality of the analysis.
